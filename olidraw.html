OliDraw

This project is currently not under development

View project here: Olidraw

Fun Facts:

  • I made this project when I was 8 with my Dad!
  • It is a cool website where you can draw anything!
This is how it works:
  1. Move your cursor to begin drawing. It will draw circles that follow your cursor with black outlines.
  2. Click + hold + move your cursor to draw with black circles.
  3. Command + R to start drawing again. (This refreshes the page)
